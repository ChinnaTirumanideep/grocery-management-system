# Grocery Inventory Management System

A modern and responsive **Grocery Inventory Management System** built with **React.js** and **JavaScript**. This application helps grocery store owners efficiently manage inventory by tracking products, monitoring stock levels, and generating inventory reports.

---

## Features

*  Interactive Dashboard
*  Add New Products
*  Edit Existing Products
*  Delete Products
*  Search Products by Name, Category, or Supplier
*  Filter Products by Category and Stock Status
*  Inventory Management
*  Low Stock & Out-of-Stock Alerts
*  Expiry Date Tracking
*  Inventory Statistics
*  Inventory Reports
*  Dark Mode Support
*  Fully Responsive Design
*  Local Storage for Data Persistence
*  Toast Notifications
*  Modern User Interface

##  Installation

### Clone the repository

```bash
git clone https://github.com/ChinnaTirumanideep/grocery-management-system.git
```

### Navigate to the project folder

```bash
cd grocery-management-system
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
